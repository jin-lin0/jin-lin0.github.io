### API

**pm.execution.setNextRequest()**

指定运行的下一个请求，仅在您运行集合时使用。

```javascript
pm.execution.setNextRequest("request_name") 
// 重复执行100次注册
for (i = 0; i <= 100; i++) { 
    console.log(i);
    pm.execution.setNextRequest("注册");
 }
pm.execution.setNextRequest(null);// 停止
```
