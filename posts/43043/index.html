typora添加自动上传图片到smms

发表于 2024-12-01 阅读次数：
本文字数： 493 阅读时长 ≈ 1 分钟

PicGo: 一个用于快速上传图片并获取图片 URL 链接的工具

PicGo 本体支持如下图床：

  • 七牛图床 v1.0
  • 腾讯云 COS v4\v5 版本 v1.1 & v1.5.0
  • 又拍云 v1.2.0
  • GitHub v1.5.0
  • SM.MS V2 v2.3.0-beta.0
  • 阿里云 OSS v1.6.0
  • Imgur v1.6.0

  1. 注册登录smms，并在API Token页面申请新的token

  2. 下载picgo软件，github链接，下载地址，mac可能不会弹出主窗口，在通知栏寻找是否有picgo的图标

  3. 进入插件设置，查看是否可以搜索到smms-user插件，笔者搜索都为空，于是采取下面的措施

  4. 打开配置文件，并记住配置文件的路径

  5. 进入该配置文件所在的文件夹，执行命令

    npm install picgo-plugin-smms-user

    注：进入Application Support文件夹的时候，使用字符串逃逸空格转义检查，即cd "Application Support"

  6. 重新回到picgo页面可以看到插件已经加载成功

  1. 进入smms-登录用户设置，输入auth即token，并设为默认图床

  2. 配置typora和配置picgo（自定义）
